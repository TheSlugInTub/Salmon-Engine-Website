A Free and Open-Source game engine
built in C++

ECS Design

The engine is built upon an ECS, where you can create entities (numbers), add components to them (structs),
and add systems (functions) to loop over components.

Physics

The engine uses Jolt Physics as it's 3D physics engine, and supports box, sphere, and capsule colliders.
