Getting started

Clone the repository by typing: `git clone https://github.com/TheSlugInTub/Salmon-Engine.git`,
although you must have git installed. If you don't have git then you can just download the source code off github.

Compiling the engine

Salmon Engine uses CMake as its build tool, you must have it installed.

Open command prompt, change directory into the folder where you downloaded the source code.
Make a build folder, change directory to the build folder, type `cmake ..` to generate build files.

Once build files are generated, change directory to the root directory and type `cmake --build build/`
to compile the engine, compiling takes a long time, have a coffee break while you're at it.

If you want to integrate the engine into your own project, copy the 'src' and 'include'
directories into your own project's source and include directories.
And copy the 'shaders' and 'res' directories into your project's root directory.
