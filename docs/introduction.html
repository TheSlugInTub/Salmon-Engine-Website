What is Salmon Engine?

Salmon engine is a game engine used for making 3D and 2D games (2D renderer not implemented yet).
It is free and open source under the GNU General Public License v3.0

These are the design goals that Salmon engine strives for:

    • Capable: It should be able to make large-scale games and offer a wide featureset
    • Simplicity: It should offer a simple yet highly versatile and infinitely powerful toolset 
    • Data-focused: It should have an entity component system paradigm 
    • Fast: It should be performant and efficient enough to run on a potato

WARNING:

Salmon Engine is extremely early in its development, important features are missing.
New versions are coming out every week.
