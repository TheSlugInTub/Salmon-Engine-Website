Here is a simple starter program with a window.

#include <iostream>
#include <salmon/salmon.h>

// Settings
const unsigned int SCR_WIDTH = 1920;
const unsigned int SCR_HEIGHT = 1080;

int main(int argc, char** argv)
{
    Window window("Window", SCR_WIDTH, SCR_HEIGHT, false /*Determines whether the window is fullscreen*/ );

    // Main scene to hold all the entities
    Scene scene;

    // Add some entities and attach some components!
    // EntityID box = scene.AddEntity();
    // scene.AssignParam<Transform>(box, glm::vec3(0.0f, 0.0f, 0.0f), glm::vec3(1.0f, 0.0f, 0.0f),
    //                             glm::vec3(1.0f, 1.0f, 1.0f));

    // Set the global engine state
    engineState.SetScene(scene); 
    // engineState.SetCamera(camera); // if you have a camera you better do this

    StartStartSystems(); // Call all the registered start systems

    // Main loop
    // -----------
    while (!window.ShouldClose())
    {
        // Call all the registered systems
        UpdateSystems();

        // Per frame logic here
        // ...

        // End of frame
        window.Update();
    }

    return 0;
}

The window class makes a window and the window.Update() at the end of the main loop swaps the buffers.

You need to set the engine state's scene as your scene because there are systems within the engine that loop over all the entities in a scene and they can't do that if they don't know what the scene is.
