Editor

Salmon Engine comes with an optional editor that you can use.
Here's how you can use it.
Instead of calling StartStartSystems on the end of the first frame, you should call StartEditorStartSystems.
And instead of calling StartSystems on every frame, you should call StartEditorSystems, and only call StartSystems if `playing` is true.
Also call DrawHierarchy, DrawInspector and DrawTray every frame. In code, it will look like this: 

//...

StartEditorStartSystems();

ImGuiLayer::Init();

// Main loop
// -----------
while (!window.ShouldClose())
{
    ImGuiLayer::NewFrame();

    UpdateEditorSystems();
    if (playing)
    {
        UpdateSystems();
    }

    DrawHierarchy();
    DrawInspector();
    DrawTray();
    //...
    ImGuiLayer::EndFrame();
    window.Update();
}

ImGuiLayer::Terminate();
//...
